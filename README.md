# Customer Segmentation & Churn Prediction using Machine Learning

## Overview

This project focuses on building an optimized Machine Learning model to predict customer churn and improve prediction performance by focusing on accuracy and precision.

The objective was to analyze customer behavior, identify important factors influencing churn, and develop a reliable classification model that can help businesses take proactive retention actions.

## Project Highlights

- Exploratory Data Analysis (EDA)
- Data preprocessing and feature engineering
- Handling imbalanced customer churn data
- Model training and performance evaluation
- Improving accuracy and precision using optimization techniques
- Customer segmentation using clustering techniques

## Machine Learning Approach

### Churn Prediction

Algorithm Used:
- Random Forest Classifier

Model improvement techniques:
- Feature selection
- Class imbalance handling
- Hyperparameter tuning
- Performance comparison

Evaluation Metrics:

- Accuracy Score
- Precision Score
- Recall
- F1 Score
- ROC-AUC Score
- Confusion Matrix


## Customer Segmentation

Technique:
- K-Means Clustering

Customer groups were identified based on customer behavior patterns to understand different customer categories and churn risks.
